# 🚀 ЗАПУСК СБОРКИ ЯДРА

## 🎯 ПРЯМАЯ ССЫЛКА ДЛЯ ЗАПУСКА:

### Способ 1: Через веб-интерфейс
1. Откройте: https://github.com/staspan34-beep/ARMNAS/actions
2. Найдите: "Build RK3566 Kernel with USB 3.0, ZFS, NPU"
3. Нажмите: "Run workflow" → "Run workflow"

### Способ 2: Прямая ссылка (API)
```
https://github.com/staspan34-beep/ARMNAS/actions/workflows/build-kernel.yml/dispatches
```

## 📋 ЧТО БУДЕТ ДЕЛАТЬ СБОРКА:

- ✅ **Armbian edge ядро** (последнее)
- ✅ **USB 3.0 поддержка** (CONFIG_USB_DWC3_HOST=y)
- ✅ **ZFS в ядре** (CONFIG_ZFS=y)
- ✅ **NPU поддержка** (CONFIG_RKNPU=y)
- ✅ **RK3566 DTB** для вашей приставки

## ⏱️ ВРЕМЯ СБОРКИ: 1-2 часа

## 📦 РЕЗУЛЬТАТ:

После сборки появятся артефакты:
- `rk3566-kernel-usb3-zfs-npu` 
- Содержит .deb пакеты для установки

## 🎯 СЛЕДУЮЩИЕ ШАГИ:

1. **Запустите сборку** сейчас
2. **Дождитесь завершения** (1-2 часа)
3. **Скачайте артефакты**
4. **Установите на NAS**
